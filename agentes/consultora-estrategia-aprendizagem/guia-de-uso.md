# Consultora de Estratégia de Aprendizagem — Guia de Uso

---

## Para que serve este agente

A Consultora de Estratégia de Aprendizagem ajuda você a **criar uma estratégia de educação corporativa alinhada com a estratégia do negócio** — definindo objetivos, competências-chave, plano de desenvolvimento e roadmap de ações por público.

Ela parte de uma recomendação estratégica concreta (idealmente vinda do Consultor de Estratégia Tecnológica) e a desdobra em um plano educacional acionável.

---

## Quando usar

- Para criar ou revisar a estratégia de T&D da sua área
- Depois de usar o Consultor de Estratégia Tecnológica e ter uma recomendação priorizada
- Para estruturar um plano de desenvolvimento para um time específico
- Para criar um roadmap de ativações de aprendizagem para o ano

---

## Passo a passo

### Passo 1 — Inicie com contexto e território

Use a recomendação copiada do agente anterior como ponto de partida:

```
"Estou analisando o contexto da empresa [NOME DA EMPRESA], 
e quero sua ajuda para analisar como podemos [RECOMENDAÇÃO 
COPIADA DO PASSO ANTERIOR] para o time [NOME DO TIME FOCO]"
```

Exemplos de times foco: liderança sênior, gerentes médios, time de tecnologia, time comercial, todos os colaboradores.

### Passo 2 — Avalie e itere

O agente vai propor uma análise inicial. Faça iterações:

- Questione competências que não pareçam aplicáveis ao seu contexto
- Ajuste o nível de maturidade atual que você percebe no time
- Peça que a análise considere restrições reais (orçamento, tempo disponível, formato preferido)

### Passo 3 — Solicite o plano de estudo

Quando estiver satisfeito com a análise de competências, peça:

```
"Agora crie um plano de estudo para o [TIME FOCO], 
a partir da maturidade que você enxerga que o time tem hoje. 
A minha métrica de sucesso será [SUA MÉTRICA]"
```

Exemplos de métricas: redução de time-to-productivity, aumento de projetos com IA em produção, melhora no eNPS, redução de turnover.

### Passo 4 — Gere o roadmap de ações

No mesmo agente, após o plano de estudo, peça:

```
"Agora crie ações de educação para:
1. Time de liderança
2. [Time foco]
3. Ações de letramento para a empresa como um todo

Se possível, organize em um roadmap para o ano com 
um calendário das ações"
```

---

## Exemplos de prompts de aprofundamento

- *"Como posso criar uma jornada de letramento em IA que não exija que as pessoas parem de trabalhar para aprender?"*
- *"Quais formatos de aprendizagem funcionam melhor para gestores com pouco tempo disponível?"*
- *"Como meço o impacto desse programa de desenvolvimento além de horas de treinamento concluídas?"*
- *"Quais são os sinais de que um programa de reskilling está funcionando — antes de ver os resultados finais?"*

---

## O que fazer com o resultado

O output deste agente é um **rascunho de estratégia educacional**. Use-o para:

1. Apresentar uma proposta inicial para as áreas de negócio parceiras
2. Iniciar uma conversa com a liderança sobre prioridades de desenvolvimento
3. Desenvolver um briefing para parceiros de treinamento e desenvolvimento
4. Criar um plano de trabalho para o time de T&D

---

## Dicas para obter melhores resultados

- **Seja honesto sobre a maturidade atual.** O agente faz escolhas melhores quando você descreve o ponto de partida real, não o ideal.
- **Especifique o público.** Um plano para 50 gestores seniores é muito diferente de um para 2.000 analistas.
- **Inclua restrições.** Se o time tem 2 horas por semana disponíveis para aprendizagem, diga isso.
- **Use a conversa de forma contínua.** Não precisa ser uma única sessão — você pode retomar a conversa e continuar de onde parou.
